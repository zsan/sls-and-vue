# Serverless Website Vue Starter Template

## Initialize
```
npx serverless init vue-starter
```

## Deploy
```
serverless deploy
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
